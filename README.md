# AI Resume Analyzer

An AI-powered Resume Analyzer built using NLP and Machine Learning.

## Features

- Resume Parsing
- ATS Score Calculation
- Job Description Matching
- Skill Extraction
- Resume Classification
- AI Feedback Generation
- Interactive Dashboard

## Tech Stack

- Python
- Streamlit
- Scikit-Learn
- Pandas
- NLP
- TF-IDF
- Cosine Similarity

## Installation

```bash
pip install -r requirements.txt
streamlit run app.py
```
